📖 About This Project

This project provides a clear, scripture-centered presentation of the gospel in one page. Each version is:

  • 🌍 Translatable — easy to adapt into any language.
  • 🖨️ Printable — one page, ready for handouts or PDF export (use the Print button).
  • 📱 Responsive — works on phones, tablets, and desktops.
  • 🙌 Open — shared under the CC0 license.
