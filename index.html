The Gospel in One Page

A simple project to explain the good news of Jesus Christ on a single page, translated into many languages.

📖 About This Project

This project provides a clear, scripture-centered presentation of the gospel in one page. Each version is designed to be:

  • 🌍 Translatable — easy to adapt into any language.
  • 🖨️ Printable — one page, ready for handouts or PDF export.
  • 📱 Responsive — works on phones, tablets, and desktops.
  • 🙌 Open — freely shared under the CC0 license.
